irbで値の表示/非表示

2007-05-24 17:47:15

すぐ忘れすのでメモ

% irb 
>> a = 100
=> 100
>> conf.echo = false
>> a
>> conf.echo = true
=> true
>> a
=> 100

たいていの場合、表示された方が便利なんだけど、
たくさんデータを持っていたり、すごく長いテキストの場合、
いちいち全部表示してると時間がかかる。
そういうデータを扱うときにfalseにしたりする。
