[ruby] MacでTwitterのページを一気にひらく適当スクリプト

2007-09-22 08:45:14
if ARGV.size == 2
  s = ARGV[0].to_i
  n = ARGV[1].to_i
  n.times do |i|
    u = "http://twitter.com/home?page=#{s + i}"
    puts "open -a Firefox #{u}"
    puts "sleep 2"
  end
end

% ruby tw.rb 12 5 |sh
