# Jottit
* 手軽でシンプル
  * トップページはテキストエリアだけ
  * 書き込むとWikiができる
* Markdown、リアルタイムプレビュー
* ユーザ登録不要 (サイトをつくるのも、編集するのも)
* プライベートなWikiはパスワードで

# 評判
> Aaron Swartzの新しいプロジェクト Jottit。 機能はごく単純なwiki (というかwikifarm) なんだけど、インタフェースが秀逸。 less is moreの精神。
